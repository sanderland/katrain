# 围棋职业比赛直播数据链调研报告

## 核心结论：数据从哪里来？

职业围棋比赛的实时棋谱数据流如下：

```
比赛现场（电子棋盘 或 人工录入）
    ↓
主办方官方中继服务器（原始源头）
    ↓
各第三方平台（星阵围棋、野狐围棋、弈客围棋等）叠加AI分析后呈现
```

**星阵围棋 (19x19.com) 不是源头**，它是下游消费者。

---

## 一、原始源头：按比赛主办方分类

### 1. 日本棋院主办的比赛 → 幽玄の間

**适用赛事：** 扇兴杯、棋圣战、名人战、本因坊战、十段战 等

- **平台：** 幽玄の間 (u-gen.nihonkiin.or.jp)
- **运营方：** 日本棋院
- **数据格式：** NGF（Nihon-Kiin Game Format）
- **访问方式：**
  - 网页：https://u-gen.nihonkiin.or.jp/live/live_list.asp
  - 专用客户端（Java桌面程序）
  - 手机APP（iOS/Android）
- **费用：** 基础账号免费注册，大型比赛（如扇兴杯）免费开放观战
- **中继日程：** https://u-gen.nihonkiin.or.jp/live/schedule_list.asp

**对于本次查询的"第8届扇兴杯世界女流最强战决赛"：**
- 原始数据源头就是 **幽玄の間**
- 赛事官方页面：https://www.nihonkiin.or.jp/special/senkocup/2026/

### 2. 韩国棋院主办的比赛 → Tygem / WBaduk / Baduk TV

**适用赛事：** 三星杯、LG杯、韩国国内棋战等

- **Tygem (弈城国际版)：** tygem.com
- **WBaduk：** wbaduk.com
- **Baduk TV：** baduk.or.kr

### 3. 中国棋院主办的比赛 → 中国棋院 / 新浪围棋

**适用赛事：** 围甲联赛、中国名人战、天元赛等

- **新浪围棋 (sinago.com)：** 国内最早的棋谱中继平台
  - 棋谱格式：**SGF（直接公开！）**
  - 棋谱URL模式：`http://sinago.com/cgibo/YYYYMM/文件名.sgf`
  - 这是**最容易程序化获取**的源头之一
- **腾讯围棋/野狐：** foxwq.com（需客户端）
- **弈客围棋：** yikeweiqi.com

### 4. 国际比赛（多方联合）

- 春兰杯、应氏杯等可能在多个平台同时中继
- 通常由举办地所在国的棋院负责原始中继

## 二、现场采集技术

比赛现场的落子如何变成数字数据：

1. **IZIS电子棋盘**（隐智科技）：安装棋子检测传感器（光感/磁感/电容），自动识别落子并记录
2. **人工录入**：由现场工作人员手动在电脑上输入每一手棋
3. **摄像头+AI识别**：较新的方式，通过摄像头拍摄棋盘并自动识别

---

## 三、程序化获取的可行性分析

| 来源 | 格式 | 公开性 | 程序获取难度 |
|------|------|--------|-------------|
| 新浪围棋 sgf | SGF | 公开URL | ★☆☆ 简单 |
| 幽玄の間 | NGF | 需注册 | ★★★ 困难（无公开API） |
| 19x19.com API | JSON/SGF | 半公开 | ★★☆ 中等 |
| 野狐围棋 | 私有 | 需客户端 | ★★★ 困难 |
| OGS (online-go.com) | JSON+SGF | 公开API | ★☆☆ 简单 |

### 推荐策略

1. **最快捷：** 直接轮询 19x19.com 的 API（虽然是下游，但接口最友好）
2. **最正统：** 注册幽玄の間免费账号，通过网页抓取获取 NGF 数据
3. **最开放：** 新浪围棋的 SGF 文件可直接 HTTP 下载
