# Tutorial Video Generation — Phase 1 POC

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Generate tutorial lecture videos from existing figure data (board_payload + narration + audio) by composing 3D Go board animation, synchronized subtitles, and voice narration into MP4 files. Phase 1 POC covers a single figure.

**Date:** 2026-04-05

## Context

The tutorial module currently displays each figure with three columns:

| Column | Title | Content | Data Source |
|--------|-------|---------|-------------|
| 1 | 原书内容 | Page image + book text | `import_book.py` |
| 2 | 棋盘识别 | 2D SVG board | `recognize_boards_v2.py` |
| 3 | 语音讲解 | Narration text + audio player | `generate_voice.py` |

We want to add **video lectures** below column 3. Each video shows the 3D Go board (reusing the game module's Three.js rendering) with stones dropping in sync with the narration audio and Chinese subtitles. No AI video generation — purely deterministic 3D rendering composed into video.

**POC target:** Section 1 (外势和实地), Figure 1 of 曹薰铉布局技巧（上）:
- 2 stones: B at [2,16] labeled "1", W at [3,3] labeled "2"
- Narration: ~130 characters, existing MP3 at `data/tutorial_assets/曹薰铉布局技巧-上册-曹薰铉-1997/audio/fig_1.mp3`
- Viewport: left half of board `{col:0, row:0, cols:10, rows:19}`

## Architecture

```
scripts/generate_video.py (Python orchestrator)
│
├─ 1. Load figure from DB (board_payload, narration, audio_asset)
│
├─ 2. Generate word-level timestamps
│     └─ edge_tts.Communicate.stream() + SubMaker
│     └─ Output: [{offset_ms, duration_ms, text}, ...]
│
├─ 3. Build timeline JSON
│     ├─ Parse move references from narration (regex)
│     ├─ Split narration into subtitle segments at punctuation
│     └─ Assign move trigger times from word timestamps
│
├─ 4. Capture video via Playwright
│     ├─ Launch headless Chrome (--use-gl=angle for WebGL)
│     ├─ context.record_video(2560x1440)
│     ├─ Navigate to http://localhost:{port}/record
│     ├─ Inject timeline via page.evaluate()
│     ├─ Recording page auto-plays: Board3D + subtitles + stone drops
│     ├─ Wait for window.__RECORDING_DONE
│     └─ Close context → saves WebM
│
├─ 5. Post-process with ffmpeg
│     ├─ Mix audio: narration.mp3 + stone sounds at trigger times
│     ├─ Merge: raw_video.webm + mixed_audio → MP4 (H.264, AAC)
│     └─ Output: 2560x1440, CRF 18
│
└─ 6. Save to data/tutorial_assets/{slug}/video/fig_{id}.mp4
```

## Key Data Structures

### board_payload (from DB)
```json
{
  "size": 19,
  "stones": {"B": [[2,16]], "W": [[3,3]]},
  "labels": {"2,16": "1", "3,3": "2"},
  "viewport": {"col": 0, "row": 0, "cols": 10, "rows": 19}
}
```

### Timeline JSON (generated by script, consumed by recording page)
```json
{
  "board_size": 19,
  "viewport": {"col": 0, "row": 0, "cols": 10, "rows": 19},
  "moves": [
    {"number": 1, "color": "B", "pos": [2, 16], "trigger_ms": 3500},
    {"number": 2, "color": "W", "pos": [3, 3], "trigger_ms": 18000}
  ],
  "subtitles": [
    {"start_ms": 0, "end_ms": 1200, "text": "在图1里，"},
    {"start_ms": 1200, "end_ms": 5500, "text": "黑棋下在1位的这手棋叫做三三点，"}
  ],
  "total_duration_ms": 28000,
  "audio_url": "/api/v1/tutorials/assets/tutorial_assets/.../audio/fig_1.mp3"
}
```

### GameState mapping (recording page converts board_payload → GameState)

Board3D requires `GameState` with `stones: [color, [col,row], scoreLoss, moveNumber][]`. The recording page progressively builds this:
- t=0: empty stones array, `current_node_index=0`
- Move 1 trigger: add `['B', [2,16], null, null]`, `current_node_index=1`, `last_move=[2,16]`
- Move 2 trigger: add `['W', [3,3], null, null]`, `current_node_index=2`, `last_move=[3,3]`

`current_node_index` increment + `last_move` change triggers the drop animation in `StoneGroup.tsx` (gravity=40, DROP_HEIGHT=5, ~0.5s duration).

```python
async def generate_audio_with_timing(text: str, audio_path: str, voice="zh-CN-XiaoxiaoNeural"):
    """Regenerate audio and capture word-level timestamps."""
    import edge_tts
    communicate = edge_tts.Communicate(text, voice)
    submaker = edge_tts.SubMaker()
    word_timings = []

    with open(audio_path, "wb") as f:
        async for chunk in communicate.stream():
            if chunk["type"] == "audio":
                f.write(chunk["data"])
            elif chunk["type"] == "WordBoundary":
                submaker.feed(chunk)
                word_timings.append({
                    "offset_ms": chunk["offset"] / 10_000,
                    "duration_ms": chunk["duration"] / 10_000,
                    "text": chunk["text"],
                })
    return word_timings
```

Note: This regenerates the audio to get timing. The regenerated audio should be identical given same text+voice. If not, we keep the regenerated version (it's the one synced to timestamps).

```python
import re

MOVE_REF_PATTERN = re.compile(
    r'(?:黑棋?的?|白棋?的?)(\d+)位?'   # 黑1, 黑棋的2位, 白棋3
    r'|(\d+)位'                          # 1位, 2位
    r'|第(\d+)[手步]'                     # 第1手, 第2步
)

def find_move_references(text: str) -> list[dict]:
    """Return [{start_char, end_char, move_number}, ...]"""
    refs = []
    for m in MOVE_REF_PATTERN.finditer(text):
        num = int(m.group(1) or m.group(2) or m.group(3))
        refs.append({"start_char": m.start(), "end_char": m.end(), "move_number": num})
    return refs
```

### Task 3: Playwright Capture + ffmpeg Composition

**File:** `scripts/generate_video.py` (continued from Task 1)

**3a. Playwright orchestration:**
```python
from playwright.async_api import async_playwright

async def capture_video(timeline: dict, port: int, output_dir: str) -> str:
    async with async_playwright() as p:
        browser = await p.chromium.launch(
            headless=True,
            args=[
                '--use-gl=angle',
                '--use-angle=swiftshader',
                '--no-sandbox',
            ]
        )
        context = await browser.new_context(
            viewport={'width': 2560, 'height': 1440},
            record_video_dir=output_dir,
            record_video_size={'width': 2560, 'height': 1440},
        )
        page = await context.new_page()

        # Navigate first, then inject data
        await page.goto(f'http://localhost:{port}/record', wait_until='networkidle')
        await page.evaluate(f'window.__RECORDING_DATA = {json.dumps(timeline)}')
        
        # Trigger the recording start
        await page.evaluate('window.dispatchEvent(new Event("startRecording"))')

        # Wait for completion with generous timeout
        timeout = timeline['total_duration_ms'] + 15000
        await page.wait_for_function(
            'window.__RECORDING_DONE === true',
            timeout=timeout
        )

        # Buffer for final animation to complete
        await page.wait_for_timeout(2000)

        await context.close()  # Saves video file
        await browser.close()

    # Find the saved WebM file
    webm_files = list(Path(output_dir).glob('*.webm'))
    return str(webm_files[0])
```

**3b. Audio mixing with ffmpeg:**

Create a mixed audio track: narration MP3 + stone placement sounds at trigger times.

```python
def mix_audio(narration_path: str, moves: list, output_path: str):
    """
    Mix narration audio with stone placement sounds.
    Stone sounds: katrain/sounds/stone{1-5}.wav (randomly picked per move).
    """
    # Build ffmpeg filter for mixing stone sounds at specific timestamps
    # adelay filter places each stone sound at the trigger time
    
    inputs = ['-i', narration_path]
    filter_parts = []
    
    for i, move in enumerate(moves):
        stone_file = f'katrain/sounds/stone{random.randint(1,5)}.wav'
        inputs.extend(['-i', stone_file])
        delay_ms = int(move['trigger_ms']) + 500  # +500ms for landing delay
        filter_parts.append(f'[{i+1}:a]adelay={delay_ms}|{delay_ms}[s{i}]')
    
    # Combine all stone sounds, then mix with narration
    stone_mix = ''.join(f'[s{i}]' for i in range(len(moves)))
    filter_parts.append(f'{stone_mix}amix=inputs={len(moves)}:normalize=0[stones]')
    filter_parts.append(f'[0:a][stones]amix=inputs=2:weights=1 0.4:normalize=0[out]')
    
    filter_complex = ';'.join(filter_parts)
    
    subprocess.run([
        'ffmpeg', '-y', *inputs,
        '-filter_complex', filter_complex,
        '-map', '[out]', '-c:a', 'aac', '-b:a', '192k',
        output_path
    ], check=True)
```

**3c. Final video composition:**
```python
def compose_final_video(webm_path: str, mixed_audio_path: str, output_path: str):
    """Merge video stream from WebM with mixed audio into MP4."""
    subprocess.run([
        'ffmpeg', '-y',
        '-i', webm_path,
        '-i', mixed_audio_path,
        '-map', '0:v',
        '-map', '1:a',
        '-c:v', 'libx264', '-preset', 'medium', '-crf', '18',
        '-c:a', 'copy',
        '-movflags', '+faststart',
        output_path
    ], check=True)
```
